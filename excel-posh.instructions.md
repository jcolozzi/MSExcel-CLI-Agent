# ExcelPOSH Module

When working with Microsoft Excel workbooks, import the PowerShell module:

```powershell
Import-Module "K:\Workgrp\PERSONAL SHARE\Colozzi\Access Agent\MSExcel-agent\ExcelPOSH\ExcelPOSH.psd1" -Force
```

This module provides 84 PowerShell functions for full Excel workbook automation via COM. Use `-AsJson` on any function for structured output. The `@excel-dev` agent has the complete function reference.
